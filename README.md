# Angular CRUD
Basic Angular CRUD.<br> The back-end is only a json-server, this project was made to train Angular framework.

## Requisites
* Node

## Instalation

In the **back-end** and **front-end** folder:
``` bash
$ npm install
```

## Running the project

In the **back-end** and **front-end** folder:
``` bash
$ npm start
```

The project will run on the  ``` localhost:4200 ```
