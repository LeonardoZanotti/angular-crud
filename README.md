# Angular CRUD
Basic Angular CRUD.

The back-end is only a json-server, this project has made to train Angular framework.

## Requisites
* Node

## Instalation
``` bash
On the /back-end folder:
$ npm install

On the /front-end folder:
$ npm install
```

## Running the project
``` bash
On the /back-end folder:
$ npm start

On the /front-end folder:
$ npm start
```

The project will run on the  ``` localhost:4200 ```
